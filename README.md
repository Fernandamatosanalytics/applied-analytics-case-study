# Applied Analytics Case Study

Business Intelligence & Applied Analytics project using Python, SQL, Power BI, and K-Means clustering.

## Project Overview
This project explores customer segmentation, business insights, KPI tracking, and dashboard development using data analytics techniques.

## Tools & Technologies
- Python
- SQL
- Power BI
- K-Means Clustering
- Data Visualization

## Key Insights
- Customer behavior segmentation
- Revenue and KPI analysis
- Business intelligence dashboards
- Data-driven decision making
